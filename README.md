vagrant-quick-mysql-server
==========================

Quickly start a clean MySQL server

    $ git clone git@github.com:keithbro/vagrant-quick-mysql-server.git

    $ cd vagrant-quick-mysql-server

    $ vagrant up

    $ mysql -h 127.0.0.1 -u root
